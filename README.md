# Lunar1ya — Библиотека игр

Персональный каталог игр с фильтрами по статусу, жанру и эпохе. Сделано для профиля [Lunar1ya](https://www.twitch.tv/lunar1ya).

## 🎮 Особенности

- **45 игр** с уникальными обложками в стиле каждой игры
- **🔐 Админка с PIN-кодом**: статусы, MP/Coop, скрытие игр, правка жанра/года — прямо на сайте
- **➕ Добавление игр с загрузкой обложек** (автосжатие до WebP 300px) — без правки кода
- **☁️ Upload для зрителей**: публикация статусов и каталога в удалённое JSON-хранилище (Pantry / jsonbin), как на [MikkleoGame](https://joonttee.github.io/MikkleoGame/)
- **💾 Экспорт / 📥 импорт** правок JSON-файлом для переноса между устройствами
- **Фильтры**: статус (все/в процессе/пройдено/в планах/не начато/MP/Coop), жанр, эпоха
- **Поиск** по названию с дебаунсом
- **Сортировка**: по названию (А-Я/Я-А) и году (новые/старые)
- **Два вида**: сетка и список
- **Статистика**: прогресс коллекции, топ жанров, счётчики
- **Адаптивный дизайн**: отдельные раскладки для телефона, планшета и десктопа
- **Оптимизированные изображения**: обложки WebP (~40-50 KB), hero/avatar в AVIF/WebP/JPG с responsive `srcset`

## 🔐 Админка и upload

В шапке сайта есть кнопка **«⚙ Админ»** (PIN-код задаётся при первом входе):

- менять статусы (⏳/✅/📌/🆕), флаги 🕹️ MP / 🤝 Coop, скрывать игры 🙈;
- править жанр и год инлайн;
- **добавлять новые игры с загрузкой обложки** прямо из браузера;
- **⬆️ публиковать правки для зрителей** в бесплатное JSON-хранилище (Pantry / jsonbin) —
  доступ к репозиторию не нужен;
- экспортировать/импортировать все правки JSON-файлом.

> Хранилище Pantry уже подключено и прописано в [`data/remote.json`](data/remote.json) —
> зрители видят опубликованное автоматически после перезагрузки страницы.

Подробная инструкция — **[ADMIN.md](ADMIN.md)**.

## 🖼️ Обложки

Все 45 обложек сгенерированы с использованием аватара Lunar1ya (серебристые волосы, фиолетовые глаза) в нативном арт-стиле каждой игры:
- Hand-drawn painterly (Constance, Hollow Knight)
- Pixel art (Stardew Valley, Replaced, Scam Line)
- Photorealistic / RE Engine (The Last of Us, Resident Evil, Metro)
- Rubber hose 1930s animation (Bendy)
- Doll-like horror (Little Nightmares)
- Low-fi UI (Scam Line)
- Storybook (Fran Bow, Creepy Tale, Moomintroll)
- И многие другие...

Обложки оптимизированы через ImageMagick: **114 MB → 1.8 MB** (63× меньше). Hero и avatar дополнительно вынесены в responsive AVIF/WebP/JPG: вместо ~5 MB PNG браузер загружает подходящий вариант примерно **6–165 KB** по размеру экрана и поддержке форматов.

## 🚀 Запуск

Просто откройте `index.html` в браузере — никаких серверов, сборок или зависимостей не требуется.

```bash
# Или через любой статический сервер
npx serve .
python -m http.server 8000
```

## 🔧 Добавление игр

**Проще всего — через админку на сайте** (⚙ Админ → «➕ Добавить игру», с загрузкой обложки).
Либо отредактируйте массив `GAMES` в `index.html`:

```javascript
const GAMES = [
  {
    title: "Название",
    genre: "Жанр",
    year: 2024,
    status: "playing",     // new | playing | done | planned
    tags: ["coop"],        // mp | coop (опционально)
    cover: "covers_webp/Name.webp",
    coverFallback: "covers/Name.png"
  },
  // ...
];
```
