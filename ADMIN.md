# 🔐 Админка и upload — Lunar1ya

Админ-панель работает по той же схеме, что и на [MikkleoGame](https://joonttee.github.io/MikkleoGame/):
правки живут локально в браузере, а кнопкой **⬆️** публикуются в удалённое JSON-хранилище,
которое подхватывают все зрители сайта. Доступа к репозиторию не требуется.

## Открытие

1. Нажмите **«⚙ Админ»** в шапке сайта.
2. При первом входе придумайте **PIN (4–6 цифр)** — он сохраняется в виде хеша только в этом браузере.
3. Сессия админа живёт до закрытия вкладки (sessionStorage). «🚪 Выйти» — завершить вручную.
4. Забыли PIN → «Сбросить» на экране входа. Правки при этом **не** удалятся.

## Что умеет админка

| Действие | Как |
|---|---|
| Статус игры | Селект в строке: `— дефолт —` / ⏳ В процессе / ✅ Пройдено / 📌 В планах / 🆕 Не начато |
| 🕹️ MP / 🤝 Coop | Кнопки-тумблеры в строке игры |
| 🙈 Скрыть от зрителей | Третий тумблер; скрытые видны только в админ-сессии (приглушены) |
| Правка жанра / года | Инлайн-поля прямо в строке игры |
| ➕ Новая игра | Форма сверху: название, год, жанр, статус, MP/Coop |
| 🖼 Загрузка обложки | Файл сжимается до 300px WebP (~30–60 КБ) и сохраняется как data-URL, либо вставьте URL картинки |
| 🗑 Удалить | Кнопка 🗑 у «своих» игр (базовый каталог не трогаем — есть 🙈) |
| 💾 Экспорт / 📥 Импорт | JSON со всеми правками и своими играми — для переноса на другое устройство |
| 🗑 Сброс правок | Удаляет все локальные правки (свои игры остаются) |

Приоритет слоёв: **локальные правки (ваш браузер) → опубликованные (remote) → базовый каталог в index.html**.

## Публикация для зрителей (upload)

> ✅ **Хранилище уже подключено.** Используется Pantry
> (`d097434f-c8ed-4b7d-b5ac-2adefd9e5231`), URL корзин прописаны в [`data/remote.json`](data/remote.json) —
> зрители подтягивают данные автоматически. Вам остаётся только нажимать **⬆️** после правок.
> При первом нажатии корзины создадутся сами.

### Как публиковать (ежедневный сценарий)

1. Откройте **⚙ Админ**, введите PIN.
2. В блоке **«☁️ Публикация»** поля уже заполнены (site подставляет их из `data/remote.json`).
   Нажмите **💾 URL** у обоих полей один раз — админка запомнит их в вашем браузере.
3. Меняйте статусы, добавляйте игры — затем:
   - **⬆️ Статусы** — публикует правки (статусы, флаги, скрытые игры, жанры/года).
   - **⬆️ Игры** — публикует весь видимый каталог (включая добавленные вами игры с обложками; скрытые не попадают).
4. Зрители увидят изменения после перезагрузки страницы сайта.

Текущие URL (не меняйте без нужды):

```json
{
  "overridesUrl": "https://getpantry.cloud/apiv1/pantry/d097434f-c8ed-4b7d-b5ac-2adefd9e5231/basket/lunar1ya-statuses",
  "gamesUrl": "https://getpantry.cloud/apiv1/pantry/d097434f-c8ed-4b7d-b5ac-2adefd9e5231/basket/lunar1ya-games"
}
```

### Если захочется другое хранилище

Замените URL в `data/remote.json` (и в админке кнопкой 💾 URL после этого — локальные URL приоритетнее).

- **jsonbin.io**: вставьте URL бина (`https://api.jsonbin.io/v3/b/…`) — заливка пойдёт методом PUT.
  Для приватного бина сохраните master-key: в консоли браузера
  `localStorage.setItem("lunar1ya_jsonbin_key", "ВАШ_КЛЮЧ")`.
- Любое другое хранилище: сайт пробует POST → PUT → PATCH и показывает понятную ошибку, если ничего не принято.

### Семантика и лимиты

Заливка **полностью заменяет** содержимое корзины (для Pantry используется POST,
т.к. PUT там делает deep-merge и «удалённые» данные бы воскресали). Лимит корзины Pantry — **1.44 МБ**
(≈ до 20–25 своих обложек поверх базового каталога).

### Приоритет источников URL (как у MikkleoGame)

`localStorage вашего браузера` → константа `REMOTE_DEFAULTS` в `index.html` → `data/remote.json`.

## Что хранится в браузере (localStorage)

| Ключ | Содержимое |
|---|---|
| `lunar1ya_overrides_v1` | Ваши правки статусов/флагов/жанров/годов |
| `lunar1ya_custom_games_v1` | Добавленные игры (обложки — data-URL) |
| `lunar1ya_admin_pin_hash_v1` | Хеш PIN |
| `lunar1ya_remote_overrides_url` | URL корзины статусов |
| `lunar1ya_remote_games_url` | URL корзины игр |

Лимит localStorage ~5 МБ на домен — сжатые обложки занимают ~30–60 КБ каждая.
